## Project Overview
This project analyzes sales performance using Excel, SQL (MySQL), and Power BI.  
The dashboard provides insights into total sales, profit, regional performance, monthly trends, product categories, and salesperson contribution.

## Tools Used
- Excel (Data Cleaning)
- Power Query
- MySQL (SQL Queries)
- Power BI (Data Visualization)

## Key Features
- KPI cards for Total Sales and Total Profit
- Sales analysis by Region
- Monthly Sales Trend
- Product Category contribution
- Salesperson performance
- Interactive slicers for region filtering
